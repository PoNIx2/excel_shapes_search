# Excel 図形検索マクロ

このリポジトリには、指定フォルダ配下の Excel ファイルを再帰検索し、
各シート内の図形を条件検索する VBA マクロを配置します。

## 1. ブック構成（標準案）

マクロ有能ブック（`.xlsm`）に以下の3シートを用意してください。

- `設定`
- `判例`
- `検索結果`

### 設定シートの入力セル

- `B2`: 検索のルートパス（必須）
- `B3`: 図形文字（任意）
- `B4`: 判例フラグ（任意: `オン` / `オフ` / `無効`）

推奨ラベル（A列）:

- `A2`: ルートパス
- `A3`: 図形文字
- `A4`: 判例フラグ

## 2. 動作仕様

- ルートパス配下をサブフォルダ含め再帰的に走査。
- 対象拡張子: `.xlsx`, `.xlsm`, `.xls`（実行ブック自体は除外）
- 各ファイルの各ワークシートの図形（Shape）を走査し、条件一致したものを結果出力。

### 条件

1. ルートパスは必須
2. 判例フラグが `無効` の場合、図形文字は必須（未入力ならエラー）
3. 判例フラグが `オン` または `オフ` の場合、`判例` シートに図形が1つ以上必要（なければエラー）
4. 判例フラグが `オン` の場合、判例図形の
   - 塗りつぶし色
   - 線色
   - 線種類（Dash）
   が一致する図形を検索
5. 判例フラグが `オフ` の場合、上記3要素が「一致しない」図形を検索
6. 図形文字が入力されている場合は、図形内テキストに部分一致するもののみ対象

## 3. 検索結果シート出力

`検索結果` シートに以下を出力:

1. ファイル名
2. シート名
3. 図形の文字

## 4. 導入手順

1. Excel で `.xlsm` ブックを作成
2. `Alt + F11` でVBEを開く
3. `ファイル > ファイルのインポート` から `src/ShapeSearch.bas` をインポート
4. `設定` / `判例` / `検索結果` シートを作成
5. `設定` シート `B2:B4` に値を入力
6. マクロ `SearchShapesInExcelFiles` を実行

## 5. 補足

- パスワード付きブックや破損ブックは読み飛ばします。
- 図形テキスト取得は `TextFrame2` 優先、取得不可時は `TextFrame` を試行します。
